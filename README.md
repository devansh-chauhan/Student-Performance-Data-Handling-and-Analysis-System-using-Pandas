# Student Performance Data Handling and Analysis System using Pandas

## Project Overview
This project uses Python and Pandas to load, clean, transform, analyze, and generate reports from a student performance dataset.

## Features
- Data Loading
- Data Cleaning
- Data Transformation
- Data Filtering
- Data Analysis
- Report Generation
- CSV Export

## Technologies Used
- Python
- Pandas
- Matplotlib
